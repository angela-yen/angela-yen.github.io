What is the difference between Padding, Border, and Margin?

So imagine you have a single wooden-framed painting on hung on a wall. You have dedicated the entire wall to this one painting. The area between the actual painting the wood-frame is the “padding”, the wood-frame itself is the border and the margin is the area outside of the border between the edge of the wall is the margin.

Padding

    The padding is the area between the content (writing, pictures etc) and the outside "frame" i.e border

    Paddings can be set for:
  • padding-top
  • padding-right
  • padding-left
  • padding-bottom

  • length - specifies a padding in px, pt, cm, etc.
  • % - specifies a padding in % of the width of the containing element
  • inherit - specifies that the padding should be inherited from the parent element

Summary

To answer the question, every webpage we see, it is divided into smaller boxes. Each "box" has 4 layers: the content, padding, border and margin. We can set specific values for each of the padding, borders and margins to align the content of have spaces between each of our boxes. Depending on where we want the space of if we want a frame, we would either change the value of the padding, border and margin.
